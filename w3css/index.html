CSS

Современный адаптивный CSS

W3.CSS - одинаково работает во всех браузерах: Chrome, Firefox, Edge, IE, Safari, Opera.

W3.CSS - одинаково работает на всех девайсах: на настольных компьютерах, ноутбуках, планшетах, смартфонах.

W3.CSS - использует только стандартный CSS (без библиотеки jQuery или JavaScript).

W3.CSS - это CSS фреймворк

W3.CSS - это современный CSS-фреймворк с поддержкой отображения по умолчанию для десктопа, планшета и мобильного устройства.

W3.CSS меньше по объёму и работает быстрее, чем аналогичные CSS-фреймворки.

W3.CSS был разработан, чтобы быть высококачественной альтернативой Bootstrap.

W3.CSS был разработан, чтобы быть меньше, быстрее, чтобы его можно было легче освоить и легче использовать.

W3.CSS также был разработан, чтобы быть независимым от jQuery или любой другой библиотеки JavaScript.

Пример

<div class="w3-container w3-teal">
  <h1>Заголовок</h1>
</div>

<img src="img_car.jpg" alt="Car">

<div class="w3-container">
  <p>Автомобиль - это самоходное транспортное средство на колесах, используемое для перевозки.</p>
</div>

<div class="w3-container w3-teal">
  <p>Футер</p>
</div>

W3.CSS бесплатный

W3.CSS бесплатный и свободный для использования. Никакая лицензия не требуется.

Как использовать W3.CSS?

Чтобы использовать W3.CSS на своем веб-сайте, просто добавьте ссылку на "w3.css" со своих веб-страниц:

Пример

<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
Попробуйте сами »

Или загрузите w3.css из w3css_downloads и добавьте ссылку на w3.css:

Пример

<link rel="stylesheet" href="w3.css">

Альтернативная ссылка на W3.CSS

Также вы можете воспользоваться альтернативной ссылкой с нашего сайта "W3Schools на русском" для подключения фреймворка W3.CSS на своем веб-сайте, просто добавив ссылку на своих веб-страницах:

Пример

<link rel="stylesheet" href="https://w3schoolsrus.github.io/lib/w3.css">

Также вы можете разместить файл w3.css на своём сайте и добавить ссылку на файл w3.css на каждой веб-странице между тегами <head> и </head>, где используется данный фреймворк:

Пример

<link rel="stylesheet" href="w3.css">
